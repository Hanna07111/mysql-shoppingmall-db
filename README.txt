This is the README.txt file for 'TEAM A'. 
It provides instructions on how to execute the code using the Runnable Jar File.


Please follow the steps below:

[1] Change connection parameter to your own sql url, root, password. It's on 18th line.
[2] Change your directory (cd) to the location where the runnable jar file is located.
[3] Run the following command: "java -jar runnable.jar"


Please make sure to replace "runnable.jar" with the actual filename of the Runnable Jar File you have. 
Execute this command to run the application.
